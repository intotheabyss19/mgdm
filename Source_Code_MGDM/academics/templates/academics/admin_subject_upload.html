Upload an Excel (.xlsx, .ods) file with columns: sub_code, sub_name, credits, and teacher_username (the username of the teacher responsible for the subject).

The teacher username should normally match their portal login username. If you only have emails, you can use the part before @ (e.g. for alice@nit.ac.in, use alice).

The subject code should follow your institute convention (e.g. CS15201), where the semester and course type (theory/lab) are encoded in the code itself. Lab subjects (code like CS15201 with a 2 in the type position) will automatically get LabInternal and LabFinal tests; theory subjects (e.g. CS15101) will automatically get Internal, Midterm1, Midterm2, and Endterm tests.
